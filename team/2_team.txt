HTML 미니 팀 과제
	OOO 업체 웹 설문지 만들기 ( 업체는 임의 )
	< 요구사항 >
	1. : 이름 입력
		ex) 김현수 
	2. : 질문(업체에 맞게 임의로) 을 5개 이상 
		ex) OOO 의료진 친절했나요? 
	3. : 질문에 대한 응답은 체크형식
		ex)   O 5점   O 4점    또는    O매우그렇다
	4. : 건의사항 [ 긴글 ]
	5. : 날짜 와 시간도 입력 
	6. : 그 외 알아서 태그나 꾸며주세요.
		- 간단한 업체 사진 
		- 폰트 적절하게 사용

	- 2. 4명이 분업( *권장 )해서 HTML/CSS 구현 		( 오늘 / 월요일 제출 )
